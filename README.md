# Reel — Movie Watchlist App

A full-stack movie browsing and watchlist app. Users can search movies, view details, and build a personal watchlist — authentication is handled with HTTP-only JWT cookies.






## Tech stack

**Frontend:** React, Vite, React Router, Tailwind CSS, Axios
**Backend:** Node.js, Express, Prisma ORM, PostgreSQL (Neon)
**Auth:** JWT stored in HTTP-only cookies
**Data:** Movie catalog seeded from [TMDB](https://www.themoviedb.org/)
